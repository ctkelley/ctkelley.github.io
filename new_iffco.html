IFFCO version 2

Version 2 of our FORTRAN implementation of implicit filtering is complete. The new code + documentation is the work of Tony Choi, Owen Eslinger, Paul Gilmore, Tim Kelley, Alton Patrick, and Joerg Gablonsky.

Implicit filtering is a finite difference quasi-Newton algorithm for noisy optimization problems with bound constraints. The method is intended for problems with optimization landscapes that look like the image above ( i.e. with many local minima).

This project is supported by the National Science Foundation.
